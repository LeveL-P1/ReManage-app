# Resident Home reference direction

## Locked scope

- Keep the existing five resident tabs: Home, Visitors, Community, Bills, and More.
- Implement only the resident Home tab in this phase.
- The supplied MyGate images are a spatial and interaction reference only. Do not use MyGate names, logos, copied advertising, or advertisement placeholders.
- All Home actions open a dedicated resident stack screen. These screens use static Expo Router paths so they remain compatible with the navigation contract.

## Home composition

1. Compact society header, kept above the scrolling feed.
2. `Quick Actions` title and `Customise` action, then a four-column, two-row grid:
   Pre-Approve, Security, Ask Society, Posts, Find Daily Help, Raise Alert, Pay Bills, and View More.
3. A rounded `You have no new updates` sheet containing `Today's Entry Updates`, a View All link, and Order now, Pre-approve, and Daily Help actions.
4. `Community Posts` with a New Post control and dense resident post cards.
5. A calm caught-up ending with an original ReManage check illustration and `View Older Posts`.

## Visual rules

- Warm neutral canvas, white cards, dark ink, teal line icons, and orange actions reuse the resident tokens.
- Use 16 px screen gutters, 58 px action tiles, 20-22 px section headings, round white cards, and compact vertical spacing to mirror the supplied reference density.
- Ads and their occupied slots are omitted. No marketing hero is introduced in their place.
- Pop-outs are full-screen resident-stack views with a close/back affordance, focused title, short explanatory copy, and a primary action where it has a real destination.
